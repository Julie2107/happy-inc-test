<h1 align="center">Тестовое задание для компании Happy Inc.</h1>

</p>
<ul>
  <li>HTML</li>
  <li>LESS</li>
  <li>SVG</li>
  <li>CSS Animations</li>
  <li>Gulp</li>
</ul>

<p> Deploy: <a href="https://fervent-villani-f4f1be.netlify.app/">https://romantic-hopper-6f4358.netlify.app/</a>
